# P1_COSC6377
Project for computer networks

Inside the JSON configuration file:
Change the Homedir to whichever place you want to place the shard in. As of now it is set to the disk space allocted to my account on program.cs.uh.edu

The steps for compiling the code are:
ant build Server Client

This  creates two executable jar files by the names NServer and NClient which need to be run using the following commands

To start server:
java -jar Nserver.jar

To start client:
Java -jar NClient.jar
